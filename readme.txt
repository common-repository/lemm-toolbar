Creates a floating toolbar on the right bottom to create link shortcuts, using font-awesome icons.

== Description ==
This Plugin will add a fixed Floating Toolbar to the right bottom corner of your Website.

In the backend you can easily add up to 8 icons for the Toolbar.
You can set a name which will be displayed on hover, an icon you want to use and a link.

This Plugin uses the following Plugins:
*Advanced Custom Field Pro
*Advanced Custom Field Font Awesome

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.tes readme files for WordPress Plugins.
